# Customer Behaviour Analysis

An end-to-end data analytics project demonstrating the complete workflow from raw data to business insights using **Python, PostgreSQL, SQL, and Power BI**.

## Project Overview

The objective of this project is to analyze customer behaviour data and uncover patterns that can support data-driven business decisions. The project covers the entire analytics process, including data exploration, cleaning, querying, and dashboard creation.

## Tools & Technologies

- **Python**
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn
- **PostgreSQL**
- **SQL**
- **Power BI**
- **Jupyter Notebook**

## Project Workflow

### 1. Data Loading and Exploration
- Imported the dataset using Python.
- Performed exploratory data analysis (EDA) to understand the data structure and identify trends.
- Examined distributions, missing values, and key variables.

### 2. Data Cleaning
- Removed duplicates.
- Handled missing values.
- Corrected data types.
- Prepared the dataset for analysis and database storage.

### 3. SQL Analysis (PostgreSQL)
- Loaded the cleaned data into PostgreSQL.
- Wrote SQL queries to answer business questions.
- Analyzed customer demographics, purchasing behaviour, and spending patterns.

### 4. Dashboard Development
- Connected Power BI to the data.
- Built an interactive dashboard to visualize key metrics and trends.
- Created charts and KPIs to support business decision-making.
